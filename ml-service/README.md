# KidneyCheck ML Service

FastAPI wrapping pre-trained CKD models from ap-atul/Chronic-Kidney-Disease (MIT).

## Setup
```bash

# 1. Install and run
pip install -r requirements.txt
python main.py
```

## Models: Naive Bayes 100%, KNN 90.62%, Logistic Regression 100%
